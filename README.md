# 📊 HR Analytics Dashboard

## 🚀 Headline
Interactive Power BI Dashboard for Employee Attrition and Workforce Analytics

---

## 📝 Short Description
The HR Analytics Dashboard is an interactive Business Intelligence solution developed using Power BI to analyze employee attrition, workforce demographics, salary distribution, and departmental performance.

The dashboard enables HR teams and business leaders to make data-driven decisions for improving employee retention, workforce planning, and organizational growth.

---

## 🛠️ Tech Stack
- 📈 Power BI Desktop
- ⚡ Power Query
- 🧮 DAX (Data Analysis Expressions)
- 📄 Excel / CSV
- 🗂️ Data Modeling

---

## 🔧 Key Technologies Used
- ⚡ **Power Query** for data cleaning and transformation.
- 📊 **DAX Measures** for KPI calculations and business metrics.
- 🔗 **Data Modeling** for building relationships between tables.
- 📉 **Interactive Visualizations** for business insights.
- 🎛️ **Slicers and Filters** for dynamic report analysis.

---

## 📂 Data Source
The dashboard uses an **Employee HR Dataset** containing:

- 🆔 Employee ID
- 👤 Employee Age
- 🚻 Gender
- 🏢 Department
- 💼 Job Role
- 💰 Salary
- 📅 Experience
- 🔄 Attrition Status

## 💡 Business Insights Examples
### 1️⃣ Workforce Overview
- 👥 Total Employees: **1,417**
- 🟢 Active Employees: **1,186**
- ⚠️ Attrition Count: **231**
- 📉 Attrition Rate: **16.3%**

### 2️⃣ Department Analysis
- 🏭 Operations department has the highest employee count.
- 💼 Employee distribution varies significantly across departments.

### 3️⃣ Salary Analysis
- 💰 The highest attrition is observed in the **6–10 LPA** salary bracket.
- 📈 Salary trends provide valuable insights into retention patterns.

### 4️⃣ Age Group Analysis
- 👨‍💼 Employees aged **26–35 years** represent the largest workforce segment.
- 📊 Workforce demographics help identify retention opportunities.

### 5️⃣ Experience Analysis
- 📅 Average employee experience is **7.04 years**.
- 📉 Attrition trends vary across different experience levels.
